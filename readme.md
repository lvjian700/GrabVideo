#Grabvideo

从Youku播放页面获取视频播放地址。该视频地址可以在VLC或者MPlayer中使用。

##依赖

* node.js: <http://nodejs.org/>
* phantomjs: <https://github.com/ariya/phantomjs>


##使用

	git clone https://github.com/lvjian700/GrabVideo.git
	cd GrabVideo
	npm install
	npm test
	npm start
